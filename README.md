# Spreadsheet Cleanup + Mini Dashboard

## What this is

I clean one messy business spreadsheet and turn it into a clean file plus a simple mini dashboard with 3-5 useful metrics.

This is a small pilot service for people who already use Excel, CSV exports, or Google Sheets to manage business data, but do not want to set up a complex tool just to understand one messy file.

## Who this is for

- Small ecommerce sellers
- Etsy / eBay / Shopify sellers
- Freelancers
- Consultants
- Small business owners
- SaaS founders managing data in spreadsheets

## Common spreadsheet problems I fix

- Inconsistent date formats
- Messy text
- Duplicate rows
- Wrong totals
- Missing values
- Inconsistent payment status
- Unnecessary notes
- Hard-to-read sales data

## Before

The messy file includes common problems found in small business exports:

- Dates appear in different formats.
- Product, category, country, and payment status values are inconsistent.
- Some fields have extra spaces.
- A customer name is missing.
- Some totals are wrong.
- Some orders are duplicated.
- Notes are inconsistent and hard to scan.

## After

The cleaned file shows what a simple cleanup pass can do:

- Dates are standardized to `YYYY-MM-DD`.
- Text fields are trimmed and normalized.
- Payment statuses are standardized.
- Totals are recalculated from `quantity x unit_price`.
- Duplicate rows are marked and excluded from dashboard totals.
- Missing values are handled clearly.
- The data is easier to read, filter, and summarize.
